# RFI-031: Cable Tray Spacing — Data Hall Corridor
**Status:** Answered  
**Discipline:** Electrical  
**Date raised:** 2025-04-18

## Question
Cable tray spacing requirement for data hall corridor routing — same as electrical room?

## Response
Same requirement as **RFI-022**: **300 mm minimum vertical spacing** between power and control trays. Refer to approved response on RFI-022 and detail E-102/A.
