# Clean Agent Fire Suppression — IT Whitespace
**Project:** Mumbai Hyperscale DC-01  
**Section:** 21 13 00  
**Standard:** NFPA 2001

## 1. Agent
Approved clean agent: **Novec 1230 (FK-5-1-12)** or approved equal. Water-based suppression not permitted in IT whitespace.

## 2. Design Concentration
Design concentration per manufacturer UL listing and NFPA 2001. Room integrity test required prior to agent release system activation.

## 3. Detection
Cross-zone detection with pre-discharge alarm and abort capability at all egress points.

## 4. Integration
Alarm integration to BMS and fire alarm panel. Supervisory signals for cylinder pressure and control panel faults.
