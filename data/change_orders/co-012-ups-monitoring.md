# Change Order CO-012: UPS Battery Monitoring Enhancement
**CO Number:** CO-012  
**Date:** 2025-04-08  
**Status:** Approved

## Description
Add cell-level battery monitoring and SNMP integration for all 500 kVA UPS modules.

## Scope
- Battery monitoring system (BMS) for each UPS string
- Integration to central DCIM platform
- Revised FAT protocol for battery data points

## References
Specification Section 26 33 16, RFI-014
